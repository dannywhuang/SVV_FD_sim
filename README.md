# SVV_FD_sim
Simulation tool for Flight Dynamics assignment SVV 2020

## How to Run
Required libraries are:
- NumPy 
- SciPy
- pandas
- matplotlib
- control

## Available functions
- calcWeightCG() 
- calcThrust()
- calcAeroCoeff()
- calcElevEffectiveness()
- calcElevDeflection()
- staticFlightCondition() 
- calcElevContrForce()
- calcEigenTest()
- stateSpace()
- calcResponse()
- plotLift()
- plotPolar()
- plotElevTrimCurve() 
- plotElevContrForceCurve()
- plotMotionsTest()
